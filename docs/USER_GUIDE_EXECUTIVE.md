## ğŸ” Mencari Dokumen

### Pencarian Cepat (Command Palette)

Cara tercepat mencari dokumen:
1. Tekan **Ctrl + K** (Windows) atau **Cmd + K** (Mac)
2. Ketik kata kunci (nomor dokumen, judul, atau isi)
3. Pilih dokumen dari hasil pencarian
4. Tekan **Enter** untuk membuka

### Pencarian dengan Filter

Untuk pencarian lebih spesifik:

1. Buka halaman **Dokumen**
2. Klik tombol **Filter** atau ikon ğŸ”

| Status | Draft, Published, Expired, dll |
| Direktorat | Dokumen dari direktorat tertentu |
| Rentang Tanggal | Dokumen dalam periode tertentu |

4. Klik **Terapkan Filter**

## â“ FAQ

### Bagaimana cara menemukan dokumen tertentu?

Gunakan pencarian cepat dengan menekan **Ctrl + K** dan ketik nomor dokumen atau kata kunci dari judul.

### Mengapa saya tidak bisa melihat beberapa dokumen?

Beberapa dokumen mungkin memiliki level kerahasiaan yang membatasi akses. Hubungi Admin jika Anda memerlukan akses ke dokumen tertentu.

### Apakah file yang saya download aman?

Ya, semua file yang diunduh memiliki watermark digital yang berisi:
- Nama pengunduh
- Tanggal dan waktu unduh
- Nomor dokumen

Ini untuk memastikan keamanan dan tracking dokumen.

### Bagaimana melihat dokumen yang sudah kadaluarsa?

1. Buka Dashboard
2. Klik angka pada widget **Kadaluarsa**
3. Atau gunakan filter status = "Kadaluarsa" di halaman Dokumen
