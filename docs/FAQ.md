## ğŸŒ Umum

### Apa itu Sistem Manajemen Dokumen Hukum?

Sistem ini adalah aplikasi berbasis web untuk mengelola dokumen hukum RS Ngoerah secara terpusat. Dokumen yang dikelola meliputi Perjanjian Kerjasama (PKS), Memorandum of Understanding (MoU), Kontrak, Surat Keputusan, dan dokumen hukum lainnya.

### Siapa yang dapat menggunakan sistem ini?

Sistem ini dapat digunakan oleh seluruh karyawan RS Ngoerah dengan level akses berbeda:
- **Super Admin/Admin** - Akses penuh untuk mengelola sistem
- **Legal Head** - Menyetujui dan me-review dokumen
- **Executive** - Melihat dan mengunduh dokumen
- **General User** - Melihat dokumen sesuai unit kerja

### Browser apa yang didukung?

Sistem ini mendukung browser modern:
- âœ… Google Chrome (rekomendasi)
- âœ… Mozilla Firefox
- âœ… Microsoft Edge
- âœ… Safari
- âŒ Internet Explorer (tidak didukung)

### Apakah sistem ini dapat diakses dari luar kantor?

Ya, sistem dapat diakses dari mana saja selama Anda memiliki koneksi internet. Pastikan Anda menggunakan koneksi yang aman dan tidak login dari perangkat publik.

## ğŸ” Login dan Akun

### Bagaimana cara mendapatkan akun?

Hubungi Admin atau bagian IT untuk pembuatan akun. Anda akan mendapatkan username dan password awal.

### Saya lupa password, bagaimana cara resetnya?

1. Hubungi Admin bagian Hukum atau IT
2. Admin akan mereset password Anda
3. Anda akan diminta membuat password baru saat login

### Mengapa saya diminta mengganti password saat login?

Ini terjadi jika:
- Ini adalah login pertama Anda
- Admin baru saja mereset password Anda
- Password Anda sudah lama tidak diganti (policy keamanan)

### Bagaimana membuat password yang kuat?

Password yang baik harus memiliki:
- Minimal 8 karakter
- Kombinasi huruf besar dan kecil
- Minimal 1 angka
- Minimal 1 karakter khusus (!@#$%^&*)
- Tidak menggunakan informasi pribadi (tanggal lahir, nama)

### Apakah akun saya bisa digunakan di beberapa perangkat?

Ya, Anda dapat login dari beberapa perangkat sekaligus. Namun, untuk keamanan, disarankan untuk logout setelah selesai menggunakan sistem, terutama di perangkat bersama.

### Akun saya terkunci, apa yang harus dilakukan?

Akun terkunci setelah 5 kali percobaan login gagal. Hubungi Admin untuk membuka kunci akun Anda.

## ğŸ“„ Dokumen

### Jenis dokumen apa saja yang ada di sistem?

- **PKS (Perjanjian Kerjasama)** - Perjanjian dengan pihak eksternal
- **MoU (Memorandum of Understanding)** - Nota kesepahaman
- **Kontrak** - Kontrak kerja dan pengadaan
- **SK (Surat Keputusan)** - Keputusan internal
- **Peraturan** - Peraturan dan kebijakan
- **Lisensi/Izin** - Dokumen perizinan
- **Dan lainnya**

### Apa arti status dokumen?

| Status | Keterangan |
|--------|------------|
| Draft | Dokumen masih dalam penyusunan |
| Menunggu Review | Dokumen sedang di-review |

| Dipublikasikan | Dokumen dapat diakses sesuai level |
| Ditolak | Dokumen perlu diperbaiki |
| Kadaluarsa | Dokumen sudah melewati masa berlaku |
| Diarsipkan | Dokumen tidak aktif tapi tetap tersimpan |

### Mengapa saya tidak bisa melihat beberapa dokumen?

Kemungkinan penyebab:
1. **Level kerahasiaan** - Dokumen bersifat confidential dan Anda tidak memiliki akses
2. **Unit kerja** - Dokumen hanya untuk unit tertentu
3. **Status** - Dokumen masih draft atau belum dipublikasikan

Hubungi Admin jika Anda memerlukan akses ke dokumen tertentu.

### Bagaimana cara melihat versi lama dokumen?

1. Buka halaman detail dokumen
2. Klik tab **Versi** atau **Riwayat Versi**
3. Anda akan melihat daftar semua versi
4. Klik pada versi yang ingin dilihat atau diunduh

### Apakah dokumen yang dihapus bisa dikembalikan?

Dokumen yang dihapus tidak langsung hilang permanen (soft delete). Admin dapat mengembalikan dokumen yang terhapus. Hubungi Admin jika perlu.

### Bagaimana cara mengunduh dokumen?

1. Buka halaman detail dokumen
2. Klik tombol **Download** ğŸ“¥
3. File akan terunduh ke folder Downloads Anda

### Mengapa file yang saya unduh memiliki watermark?

Watermark adalah fitur keamanan untuk:
- Melacak siapa yang mengunduh dokumen
- Mencegah penyalahgunaan dokumen
- Memastikan keaslian dokumen

Watermark berisi: nama pengunduh, tanggal/waktu unduh, dan nomor dokumen.

### Apakah saya bisa mengunduh dokumen tanpa watermark?

Tidak. Semua dokumen yang diunduh akan memiliki watermark untuk alasan keamanan.

### Bagaimana cara mencetak dokumen?

1. Buka halaman detail dokumen
2. Klik tombol **Cetak** ğŸ–¨ï¸

1. Anda tidak memiliki permission download
2. Dokumen masih berstatus draft
3. File dokumen rusak atau tidak tersedia

Hubungi Admin jika masalah berlanjut.

## ğŸ” Pencarian

### Bagaimana cara mencari dokumen dengan cepat?

Tekan **Ctrl + K** (Windows) atau **Cmd + K** (Mac) untuk membuka pencarian cepat, lalu ketik kata kunci.

### Apa saja yang bisa dicari?

Anda dapat mencari berdasarkan:
- Nomor dokumen
- Judul dokumen
- Deskripsi/ringkasan
- Kata kunci dalam konten dokumen
- Nama pihak terkait

### Bagaimana cara menggunakan filter?

1. Buka halaman **Dokumen**
2. Klik ikon filter atau panel filter di sebelah kiri
3. Pilih kriteria:
   - Jenis dokumen
   - Kategori
   - Status
   - Tanggal
   - Unit kerja
4. Klik **Terapkan**

### Pencarian tidak menemukan hasil, apa yang harus dilakukan?

Coba:
1. Periksa ejaan kata kunci
2. Gunakan kata kunci yang lebih umum
3. Hapus beberapa filter
4. Gunakan wildcard (*)
5. Cari dengan nomor dokumen jika tahu

## ğŸ”§ Teknis

### Halaman tidak loading/blank, apa yang harus dilakukan?

1. Refresh halaman (Ctrl + F5 atau Cmd + Shift + R)
2. Clear cache browser
3. Coba browser lain
4. Periksa koneksi internet
5. Hubungi Admin jika masalah berlanjut

### Sistem lambat, apa penyebabnya?

Kemungkinan penyebab:
- Koneksi internet lambat
- Banyak tab browser terbuka
- Cache browser penuh
- Server sedang sibuk (peak hours)

Solusi:
- Tutup tab yang tidak diperlukan
- Clear cache browser
- Coba akses di luar jam sibuk

### Error 403 Forbidden

Artinya Anda tidak memiliki izin untuk mengakses halaman tersebut. Hubungi Admin jika merasa seharusnya memiliki akses.

### Error 404 Not Found

Halaman tidak ditemukan. Kemungkinan:
- URL salah
- Dokumen sudah dihapus
- Link sudah tidak valid

### Error 500 Internal Server Error

Terjadi kesalahan di server. Laporkan ke Admin dengan informasi:
- Halaman yang diakses
- Waktu kejadian
- Aktivitas yang sedang dilakukan

### Bagaimana cara membersihkan cache browser?

**Chrome:**
1. Ctrl + Shift + Delete
2. Pilih "Cached images and files"
3. Klik "Clear data"

**Firefox:**
1. Ctrl + Shift + Delete
2. Pilih "Cache"
3. Klik "Clear Now"

**Safari:**
1. Command + Option + E
2. Atau: Safari > Clear History
